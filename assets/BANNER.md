# Banner

The STACK boot banner. Show it when STACK activates (terminal, chat, or otherwise). It must render **at the active size**, so:

- **Always print it inside a fenced code block** so the monospace grid holds and the surface scrolls instead of wrapping it into noise.
- **Wide surface** (full terminal, desktop chat): full art below.
- **Narrow surface** (mobile chat, slim pane): compact wordmark below.
- **In a terminal**, run `bin/banner.mjs` — it auto-detects width, picks the right tier, and colors it (teal wordmark, grey band) on a TTY. Color is terminal-only; it does not show inside a chat code block.
